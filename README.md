## Description

Get the ball in the other player's goal. First to 11 wins. Balls hit out of bounds don't affect the score. Requires 2 players.

## Controls

```yaml
Game:
  Exit: Escape
  Select: Space
  Restart: R
Player 1:
  Move Up: W
  Move Down: S
  Serve: A
Player 2:
  Move Up: Up Arrow
  Move Down: Down Arrow
  Serve: Right Arrow
```
